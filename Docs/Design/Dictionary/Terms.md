# Terms Dictionary 
- WF - aka Workflow - is a set of HTTP tasks, VMAs and CWF that are excuted in a certain order 
- CWF - Child Workflow is aka Nested Workflow (NFW) - is a workflow that is being exeucted by another Workflow 
- PWF - Parent Workflow is a workflow that executes a CWF 
- CCWF - Is a CWF of a CWF of a PWF 
- CCCWF - is a CWF of a CWF of a CWF of a PWF 
- WFA - The Workflow Admin shelf of the Workflow that is can be opened from the workflow designer. 
- EWF - Event Workflow - a Workflow triggered as an Event Trigger that is defined in the WFA 
- WFM - Is the Workflow Meta shelf opened from the WF Inspect 
- WFI - is Workflow Inspect Shelf 
- VMA - Variables Manipulation Activity - is a task that is used to manipulate variables in a workflow
- WFD - Workflow Designer 

# Scheuling

- CAL - Calendar 

# Workflow Utilities 
- IFN - If node - is a node that is used to evaluate a condition and return a value based on the condition
- VMA - Variables Manipulation Activity - is a node that is used to manipulate variables in a workflow
